# The Conference Room Scheduler

A Python-based GUI application for scheduling and managing conference rooms efficiently.

## Overview
This project provides a graphical interface to easily book and manage available conference rooms.

## Files included
- **`guiNew.py`**: The main GUI application script for the scheduler.
- **`The-Conference-Room-Scheduler (1).pdf`**: Project documentation outlining the requirements, design, and details of the conference room scheduler.

## Prerequisites
- Python 3.x installed on your system.
- Standard GUI libraries (like `tkinter`, typically included with standard Python installations) or any specific dependencies required by `guiNew.py`.

## How to Run
Execute the Python script to start the scheduler application:
```bash
python guiNew.py
```
