# The Conference Room Scheduler

## Overview
The Conference Room Scheduler is a Python-based graphical user interface (GUI) application designed to manage and schedule conference rooms efficiently. It uses advanced Data Structures and Algorithms, specifically a **Van Emde Boas (vEB) Tree**, to handle room availability, booking conflicts, and resource management with high performance.

## Features
* **Intuitive GUI:** Easy-to-use graphical interface built with Tkinter.
* **Room Management:** View available conference rooms and their details.
* **Scheduling:** Book rooms for specific time slots, avoiding double-bookings.
* **Smart Booking:** Automatically find and book the first available slot.
* **VEB Tree Visualization:** Visual representation of the underlying data structure managing the schedule.
* **Conflict Resolution:** Efficiently checks for scheduling conflicts.

## Technologies Used
* **Language:** Python
* **Libraries:** Tkinter (GUI), datetime
* **Key Concept:** Van Emde Boas Tree (Data Structures and Algorithms)

## How to Run
1. Ensure Python 3 is installed on your system.
2. Navigate to the project directory.
3. Run the main Python script using:
   ```bash
   python "room scheduler.py"
   ```
